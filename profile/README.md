## ğŸ¯ What is Walmarket?

Walmarket is the world's first **verifiable AI oracle for prediction markets**, combining Nautilus TEE, GPT-5, and SUI blockchain to create trustless, cryptographically-proven market resolution.

Built for **Walrus Haulout Hackathon 2025** (Provably Authentic track), Walmarket eliminates the need to trust oracle operators by running AI inference inside secure enclaves with remote attestation.

### ğŸ” Why Verifiable?

Traditional AI oracles require blind trust. Walmarket uses:

- **Nautilus TEE Integration**: AI inference in secure enclaves with cryptographic attestation
- **GPT-5 Verification**: Predictions verified through TEE signatures and on-chain validation
- **Walrus Storage**: Complete audit trail (input, output, reasoning) stored immutably
- **On-Chain Proof**: SUI smart contracts validate TEE attestation and enforce slashing
- **No Trust Required**: Cryptographic proofs guarantee honest execution
